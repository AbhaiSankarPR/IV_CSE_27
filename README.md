# IV Essentials & Dashboard

A modern, responsive dashboard web app for managing **Industrial Visit (IV)** essentials, itinerary, coordinators, and image uploads. Built with **React** and **Tailwind CSS**, with authentication and interactive features.

---

## Features

### 1. User Profile Panel
- Displays logged-in user's name and email.
- Shows a welcome message: *"You are successfully logged in. Enjoy the journey!"*  

### 2. IV Essentials Checklist
- Categorized checklist:
  - Clothing
  - Toiletries & Grooming
  - Medicines / Health
  - Other Essentials
  - General Guidelines
- Styled cards with backdrop blur and modern design.  

### 3. Image Uploads
- Separate sections for **Personal** and **Public** uploads.
- Supports selecting multiple images.
- Designed for easy backend integration in the future.

### 4. Travel Itinerary & Trips
- View **full itinerary** including boarding, hotel stays, and sightseeing events.
- Each day lists all planned activities for easy reference.

### 5. Map & Location Features
- Map showing locations of trip destinations.
- **Get directions** to admin or key locations.
- Option to **track your own location** during travel.

### 6. Coordinators Information
- List of coordinators with name, role, and contact details.
- Easily accessible for queries or assistance.

### 7. Responsive Design
- Sidebar collapses on mobile.
- Vertical menu for desktop; mobile-friendly layout for all panels.

### 8. Logout Button
- Accessible at the bottom of the sidebar for quick logout.

---

## Tech Stack

- **React.js** – Frontend UI  
- **Tailwind CSS** – Styling  
- **Context API** – Authentication management  
- **Optional:** Google Maps API for map and directions
