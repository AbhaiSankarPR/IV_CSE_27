# IV Live

A web app built for the **Industrial Visit 2025** of the  
**Computer Science & Engineering Department (Batch 2023–27)**,  
SCT College of Engineering, Trivandrum.

## Tech Stack
- React + Vite  
- React Router DOM for navigation  
- Custom CSS styling

## Features
- Home page summarizing the IV journey  
- Coordinators section with team details  
- Planned pages: Schedule, Images, and more

## Route
Ahmedabad → Jaisalmer → Jaipur → Shimla → Delhi
